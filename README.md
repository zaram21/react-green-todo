# Advanced Todo App with React + Framer Motion + TailwindCSS

یک Todo App پیشرفته و چشم‌نواز ساخته‌شده با React, Framer Motion, TailwindCSS و Lucide Icons.  
این پروژه شامل انیمیشن‌های نرم، طراحی مدرن با رنگ سبز زمردی، و ذخیره‌سازی محلی (LocalStorage) برای پایداری داده‌هاست.

---

## ویژگی‌ها

-  افزودن، حذف و تغییر وضعیت تسک‌ها با انیمیشن
-  ذخیره‌سازی خودکار در LocalStorage
-  فیلتر بر اساس وضعیت (All / Active / Completed)
-  طراحی مدرن با TailwindCSS و پالت Emerald Green
-  ترنزیشن نرم و انیمیشن با Framer Motion
-  رابط کاربری مینیمال و تجربه کاربری بالا
-  فونت‌های خاص و رایگان (Poppins & Nunito)


## تکنولوژی‌های استفاده‌شده

| تکنولوژی | توضیح |
|  React | ساختار اصلی پروژه |
|  TailwindCSS | طراحی سریع و مدرن UI |
|  Framer Motion | انیمیشن و ترنزیشن زیبا بین 
|  Lucide Icons | آیکون‌های سبک و مدرن |
|  Context API + Custom Hook | مدیریت وضعیت تسک‌ها و LocalStorage |
